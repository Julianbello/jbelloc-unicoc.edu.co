Que es topologia de red?

La topología de red es la disposición de los elementos de una red de comunicación, se puede utilizar para definir o describir la disposición de varios tipos de redes de telecomunicaciones.



Hub

Es un dispositivo que permite conectar equipos y dispositivos en red. Esa red puede ser local (LAN) y mantenerla así o conectarla a su vez a Internet.



Acces Point

Es un dispositivo de red que permite que los dispositivos con capacidad inalámbrica se conecten a una red cableada.



Wireless

Es la transmisión de voz y datos sin cables ni alambres. En lugar de una conexión física, los datos viajan a través de señales electromagnéticas transmitidas desde las instalaciones de envío hasta los dispositivos intermedios y del usuario final.



DHCP

Es un protocolo de administración de red que se utiliza para asignar dinámicamente una dirección IP a cualquier dispositivo o nodo en una red para que pueda comunicarse mediante IP.



DNS

Un sistema de nombres de dominio (DNS) convierte los nombres de dominio en direcciones IP, que permiten a los navegadores acceder a sitios web y otros recursos de Internet.



Bus

Un bus es un sistema de comunicación que transfiere datos entre componentes dentro de una computadora o entre computadoras. Es un conjunto de conductores paralelos o un conjunto de conductores agrupados entre sí para formar un cable.



Backbone

Parte de una red que actúa como ruta primaria para el tráfico que, con mayor frecuencia, proviene de, y se destina a, otras redes.



Fibra óptica

Tipo de cable de red que utiliza delgados filamentos de vidrio para transportar información digital que ha sido transformada en impulsos de luz.



Encapsulado de datos

Proceso en el que un protocolo del host de envío agrega datos al encabezado del paquete.



Host

Son los computadores conectados a la red, que proveen o utilizan servicios de ella



Nodos

Un nodo es una colección de servidores de la plataforma de BI que se ejecutan en el mismo host y los gestiona un solo Agente de inteligencia de servidor.



Adaptador de red

Dispositivo que recibe y convierte señales entrantes de la red a la estación de trabajo y convierte y envía comunicaciones salientes a la red.



Enrutador

Dispositivo que se utiliza para conectar dos o más redes y enrutar el tráfico de red entre ellas.



Switch

Dispositivo que se utiliza para conectar varios dispositivos en una red y enrutar el tráfico de red entre ellos.



Firewall

Dispositivo que se utiliza para proteger una red de amenazas externas, como virus y ataques de hackers.



VLAN

Red de área local virtual que se utiliza para separar el tráfico de red en diferentes segmentos lógicos.



Gateway

Dispositivo que se utiliza para conectar dos redes diferentes y permitir que los dispositivos de una red se comuniquen con los dispositivos de la otra red.



Puente

Dispositivo que se utiliza para conectar dos segmentos de red y permitir que los dispositivos en cada segmento se comuniquen entre sí.
